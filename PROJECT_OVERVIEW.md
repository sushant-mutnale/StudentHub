# StudentHub – AI-Powered Career Development Platform
**Empowering your career journey with personalized AI insights, skill analysis, and recruitment tools.**

---

## 2. Project Purpose
StudentHub was created to bridge the gap between academic learning and industry requirements. It aims to solve the problem of "employability" by providing students with a clear, AI-driven path to success, while giving recruiters a high-precision tool to find the right talent. Unlike existing systems that are often fragmented (learning on one site, jobs on another), StudentHub integrates the entire career lifecycle into a single, intelligent ecosystem.

## 3. Problem Statement
The current career landscape presents several hurdles:
*   **Skill Gaps**: Students often don't know exactly what skills they are missing for a specific role.
*   **Resource Overload**: High-quality learning resources are scattered, making it hard to follow a structured path.
*   **Recruitment Friction**: Recruiters struggle to filter through thousands of resumes to find candidates who truly match their needs.
*   **Lack of Preparation**: Students lack affordable, high-quality interview and coding practice tools tailored to real job descriptions.

## 4. Why This Project Matters
StudentHub matters because it democratizes career coaching.
*   **Personalization**: Every student gets a unique learning roadmap.
*   **Job Readiness**: It moves students beyond theoretical knowledge into interview-ready professionals.
*   **Efficiency**: Recruiters save time by viewing AI-matched profiles instead of raw resumes.
*   **Modern Alignment**: In today's fast-paced tech market, staying updated on specific industry "tech stacks" is crucial, and StudentHub automates this tracking.

## 5. High-Level System Overview
StudentHub is an end-to-end platform with two primary entry points:
*   **Student Dashboard**: A hub for skill analysis, roadmaps, coding practice, and job applications.
*   **Recruiter Dashboard**: A command center for posting jobs, managing the hiring pipeline, and viewing AI-recommended candidates.
*   **The AI Core**: The "brain" of the system that connects these two sides by analyzing resumes, job descriptions, and user performance to ensure the best possible match.

## 6. Main Modules of the System
*   **User & Profile Management**: Complete registration system with secure authentication, multi-role support, and advanced AI-parsed profile management.
*   **Skill Gap Analysis**: Deep AI analysis of current skills against target industry roles to find specific "missing" competencies.
*   **Learning Path Generator**: An automated roadmap generator that builds structured modules with quizzes and progress tracking.
*   **Opportunities & Job Matching**: A smart recommendation engine that suggests jobs based on the student's current skill level and learning progress.
*   **Deep Research**: Tools to research companies, culture, and interview processes before applying.
*   **Interview Preparation**: Includes text-based AI chat interviews and advanced **AI Voice Interviews** for a realistic simulation.
*   **DSA Coding Practice**: A built-in coding environment supporting multiple languages with automatic evaluation.
*   **Notifications System**: Real-time alerts for jobs, interviews, and messages.
*   **Social Feed**: A professional networking wall to share achievements, posts, and connect with peers.
*   **Recruiter Dashboard**: Advanced tools for job posting, automated candidate matching, and a visual hiring pipeline.

## 7. System Workflow
1.  **Onboarding**: Student registers and creates a profile.
2.  **Analysis**: Student uploads a resume; the AI parses it and performs a **Skill Gap Analysis**.
3.  **Growth**: A **Learning Path** is generated. The student completes modules and practices coding.
4.  **Preparation**: The student uses **AI Voice Interviews** and **Deep Research** to prepare for specific companies.
5.  **Matching**: The system identifies perfect Job Matches.
6.  **Recruitment**: Recruiters post jobs, the AI suggests students, and interviews are initiated directly through the platform.

## 8. Key Technologies Used
*   **Frontend**: React.js with Vite for high performance and Vanilla CSS for a premium, custom UI/UX experience.
*   **Backend**: Python with **FastAPI** for high-concurrency, asynchronous API performance.
*   **Database**: **MongoDB** for flexible, document-based storage to handle complex profiles and feeds.
*   **Real-time**: Redis for event handling and caching.
*   **AI Components**: Advanced LLM integration for resume parsing, skill gap analysis, and interactive AI Interviewers.
